FAQ

Часті запитання

Я ніколи не працювала з кодом. Чи впораюсь?
Так. Воркшоп побудований саме для людей без технічного досвіду. Claude Code пише код за тебе — ти описуєш задачу словами.
Що мені потрібно підготувати заздалегідь?
Claude Pro ($20/міс), Anthropic API key ($5), Telegram бот, GitHub акаунт і VS Code. Детальна інструкція надходить після реєстрації.
Скільки коштує підтримка агента щомісяця?
VPS $6/міс + Anthropic API (орієнтовно $5–15/міс). Разом ~$20–25/міс.
Чи буде запис воркшопу?
Запису не буде — це камерний живий формат де важлива твоя присутність і практика.
Що якщо щось не вийде під час воркшопу?
Ми працюємо в малій групі — у кожного буде підтримка. Плюс закритий чат де можна дописати після.
Коли наступний воркшоп?
Дата буде оголошена після закриття поточної групи. Early bird діє до заповнення місць.
